ОТПРАВКА ДАННЫХ НА ВХОД

 Если проверка логин/пароль успешна, возвращаем error = 0 и происходит перезагрузки страницы
 Если не зарегистрирована почта или неверный пароль, возвращаем error = 1 и error_text = пояснение к ошибке

ВОССТАНОВЛЕНИЕ ПАРОЛЯ

 Если почта существует и пароль отправлен, возвращаем error = 0
 Если не зарегистрирована почта, возвращаем error = 1 и error_text = пояснение к ошибке
 
РЕГИСТРАЦИЯ 

 Если успешно зарегистрирован, возвращаем error = 0
 Если почта уже зарегистрирована или пароли не совпадают, возвращаем error = 1 и error_text = пояснение к ошибке

ДОБАВЛЕНИЕ В СРАВНЕНИЕ

  Если успешно добавлено, возввращаем compare_total = количество позиций в сравнении

ДОБАВИТЬ В КОРЗИНУ C ГЛАВНОЙ ИЛИ КАТАЛОГА

 Если успешно добавлено, возввращаем basket_total = количество позиций в корзине

ФИЛЬТР ТОВАРОВ

 При успешном получении данных из фильтра возвращаем:
 counter = количество получившихся позиций
 content = разметка подгружаемого содержимого
 has_load = актуальна ли ссылки "подгрузить еще" 1 - да, 0 - нет
 load_link = текст для ссылки. "Показать еще N (Осталось M)"

ПОДГРУЗКА ТОВАРОВ

 Аяксом отправляется количество уже загруженных элементов в списке.
 Возвращаемые данные как в фильтре, кроме отсутсвующего поля counter

ИЗМЕНЕНИЕ КОЛИЧЕСТВА ТОВАРОВ В КОРЗИНЕ

 total - сумма итогова по корзине
 total_el - итог по позиции

УДАЛЕНИЕ ПОЗИЦИИ В КОРЗИНЕ

 basket_total - количество товаров в козине в шапке
 total - сумма итогова по корзине

ДОБАВЛЕНИЕ ПОЗЦИИ В КОРЗИНУ СО СТРАНИЦЫ КОРЗИНЫ ИЗ РАЗДЕЛА СРАВНЕНИЕ

 basket_total - количество товаров в козине в шапке
 total - сумма итогова по корзине
 added_content - разметка добавляемой позиции

ПЕРЕКЛЮЧЕНИЕ РЕГИОНА ДОСТАВКИ В КОРЗИНЕ ДОСТАВКА И САМОВЫВОЗ

 content - разметка добавляемых способов доставки
 в data к картам указать новые метки
